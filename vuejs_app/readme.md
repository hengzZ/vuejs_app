##### 1. Node.js 运行环境

##### 2. vue-cli 工程搭建助手
```
# 全局安装 vue-cli
npm install --global vue-cli
# 创建一个基于 webpack 模板的新项目
vue init webpack my-project
# 安装依赖， 走你
cp my-project
npm install 
npm run dev
此时，你已拥有一个 Vue 的 demo 项目实例
```
